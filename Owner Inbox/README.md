# Owner Inbox

This is where the team delivers completed work to the owner.

## Purpose
Team members place finished outputs, reports, and results here for the owner to review.

## How It Works
1. A team member completes their assigned task
2. They deliver the result here as a markdown file or attachment
3. The owner reviews completed work in this folder

## What Gets Delivered Here
- Completed research reports
- Hired agent profiles
- Task outputs and summaries
- Status updates
